
# go-faas-cloud-simulator

Cloud FaaS Workload Simulator written in Go using the [Azure Functions Trace from 2019](https://github.com/Azure/AzurePublicDataset/blob/master/AzureFunctionsDataset2019.md).
## Installation

Build the project

```bash
  $ ./build.sh
```
## How to run

To run the project:

```bash
  $ ./sim
```
